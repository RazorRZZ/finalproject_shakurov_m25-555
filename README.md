Итоговый проект "Платформа для отслеживания и симуляции торговли валютами" - это инициализируемая программа, имитирующая работу действующего примитивного приложения по трейдингу, выполненная в рамках освоения дисциплины "Программирование на Python" студента группы М25-555 Шакурова Егора.

Инструкция по сборке проекта и запуску программы:

1. Установка зависимостей: make install
2. Установка необходимых пакетов: make package-install
3. Запуск программы (платформы): make project
4. Сборка проекта: make build
5. Публикация сборки: make publish
6. Проверка кода в соответствии с ruff: make lint
